# WebReportV9Exp
WebReport V9 Poc &amp; Exp
# Usage 
```
optional arguments: 
  -h, --help  show this help message and exit 
  -u U        目标URL,如:http://test.com 
  -f F        目标URL文件 
  -p P        代理地址,如:http://proxy.com 
  -sf SF      需要上传到目标服务器的文件,如:shell.jsp 
  -o O        保存结果到文本文件 
```
